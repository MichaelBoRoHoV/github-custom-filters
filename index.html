Privacy Policy

GitHub Custom Filters Extension

Version 1.0

Data Collection and Usage

This extension does not collect, transmit, or share any personal data with external servers. Your privacy is our priority.

Local Storage

The extension uses Chrome's local storage to save your custom filter configurations. This data includes:

  • Filter names you create (e.g., "My PRs", "Bug Reports")
  • GitHub search queries you save (e.g., "is:pr author:@me")

Data Location

All data is stored locally on your device and is not transmitted to any external servers. If you have Chrome sync enabled, this data may be synchronized across your other Chrome browsers, but it remains under your control.

Third-Party Access

No third parties have access to your data. The extension only interacts with GitHub.com to provide the filtering functionality.

Data Deletion

You can delete your data at any time by:

  • Removing individual filters through the extension popup
  • Uninstalling the extension (which removes all stored data)

Contact Information

If you have questions about this privacy policy, please contact the developer through the Chrome Web Store listing.

This privacy policy is effective as of the date listed below and will remain in effect except with respect to any changes in its provisions in the future.
